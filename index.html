WHY SNAP-SOLVER

为部署在自己电脑上而设计

局域网直连

手机与电脑同一 Wi-Fi 即可使用，截图不经任何中间服务器，回传快、不排队。

六家模型任选

Claude、GPT、Gemini、通义、豆包、Kimi 同屏切换，哪个顺手用哪个，还能换模型重解对比。

思考过程可见

推理模型「怎么想的」以淡色小字实时展开，答完自动收起——看过程比抄答案更有用。

同题追问

答案没看懂，就在底部输入框继续问。上下文 = 题目截图 + 已有问答，换题或返回即清空。

自带密钥，数据自持

用你自己的 API Key 直连各厂商，密钥只存在电脑本地的 .snapsolver/，没有账号、没有订阅。

全平台，免安装

电脑端支持 Windows / macOS / Linux；手机端就是浏览器，不用装 App、不用注册。

MODEL LINEUP

接入六家厂商的当家模型

统一 Fast / High / Max 三档推理档位，切换模型时按能力自动适配；国内厂商无需代理即可直连。
